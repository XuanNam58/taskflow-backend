# Taskflow Backend

Taskflow is a SaaS-style task management backend built with Spring Boot, PostgreSQL, Flyway, Spring Security, JWT, and JPA.

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Flyway
- Spring Security
- JWT
- MapStruct
- Lombok

## Current Features

- User entity
- Refresh token entity
- Flyway database migration
- PostgreSQL configuration

## Run Locally

```bash
mvn clean install
mvn spring-boot:run
